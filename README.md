# Pantry Inventory

A small personal pantry inventory app with Google Sheets sync.

## Files

- `index.html` - app page
- `assets/styles.css` - visual design
- `assets/app.js` - app logic
- `google-apps-script.js` - Google Apps Script backend for Sheets sync
- `GOOGLE_SHEETS_SYNC.md` - setup notes

## GitHub Pages

This app can be hosted as a static site with GitHub Pages. The pantry data is stored in Google Sheets when sync is configured.
